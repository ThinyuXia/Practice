#### 一、质数

##### 1.试除法判定质数

```java
public static boolean check(int n){
  if(n < 2) return false;
  for(int i = 2;i <= n / i;i ++){
    if(n % i == 0) return false;
  }
  return true;
}
```

##### 时间复杂度：O(sqrt(n))  

##### 空间复杂度： O(1)



##### 2.拭除法求质因数

```java
import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int T = in.nextInt();
        while(T -- != 0){
            int n = in.nextInt();
            for(int i = 2;i <= n / i;i ++)
              //1.能整除n的i一定是质数
              //理由：枚举到i时，n中2到i - 1之间的质因子已经被删除完毕，因为i是n的因子，但是n中不包含2到i - 1之间的质因子，所以i一定是质数
                if(n % i == 0){
                    int s = 0;
                    while(n % i == 0){
                        n /= i;
                        s ++;
                    }
                    System.out.println(i + " " + s);
                }
            if(n != 1) System.out.println(n + " " + 1);    
            System.out.println();
        }
    }
}
```
