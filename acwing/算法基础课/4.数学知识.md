#### 一、质数

##### 1.试除法判定质数

```java
public static boolean check(int n){
  if(n < 2) return false;
  for(int i = 2;i <= n / i;i ++){
    if(n % i == 0) return false;
  }
  return true;
}
```

##### 时间复杂度：O(sqrt(n))  

##### 空间复杂度： O(1)



##### 2.拭除法求质因数

```java
import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int T = in.nextInt();
        while(T -- != 0){
            int n = in.nextInt();
            for(int i = 2;i <= n / i;i ++)
              //1.能整除n的i一定是质数
              //理由：枚举到i时，n中2到i - 1之间的质因子已经被删除完毕，因为i是n的因子，但是n中不包含2到i - 1之间的质因子，所以i一定是质数
                if(n % i == 0){
                    int s = 0;
                    while(n % i == 0){
                        n /= i;
                        s ++;
                    }
                    System.out.println(i + " " + s);
                }
            if(n != 1) System.out.println(n + " " + 1);    
            System.out.println();
        }
    }
}
```



##### 2.1埃氏筛法

```java
// 思路证明：为什么删除完的数组中所有数均为质数，反证法证明：若删除完的数组中
// p不是质数，则它一定是2 ～ (p - 1) 中某一个质数的若干倍，但是2 ～ (p - 1)中的
// 的每一个质数的倍数已经删完了，所以p一定是质数，证毕。
import java.util.*;
public class Main{
    
    static int[] p = new int[1000010];
    static boolean[] s = new boolean[1000010];
    static int cnt = 0;
    static void get_primes(int n){
        for(int i = 2;i <= n;i++){
            if(!s[i]) {
                p[cnt++] = i;
                for(int j = i;j <= n;j += i) s[j] = true;
            }
        }
        System.out.print(cnt);
    }
    
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        get_primes(n);
    }
}
```



##### 2.2线性筛法

```java
//核心思想：每个合数都被他的最小质因子筛掉
import java.util.*;
public class Main{
    static int[] primes = new int[1000010];
    static boolean[] s = new boolean[1000010];
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        int cnt = 0;
        for(int i = 2;i <= n;i ++){
            if(! s[i]) primes[cnt ++] = i;
            for(int j = 0;i * primes[j] <= n;j ++){
                s[i * primes[j]] = true;
                if(i % primes[j] == 0) break;
            }
        }
        System.out.println(cnt);
    }
}
```

##### 2试除法求约数

```java
 //朴素法枚举
import java.util.*;
public class Main{
    
    public static void get_divisors(int n){
        ArrayList<Integer> a = new ArrayList<>();
        int cnt = 0;
        for(int i = 1;i <= n / i;i ++){
            if(n % i == 0) 
            {
                a.add(i);
                if(n / i != i) a.add(n / i); 
            }
        }
        Collections.sort(a);
        for(int x : a) 
            System.out.print(x + " ");
        System.out.println();
    }
    
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int T = in.nextInt();
        while(T -- != 0){
            int n = in.nextInt();
            get_divisors(n);
        }
    }
}
```
